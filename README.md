## 📋 Overview

LifeLine is a comprehensive, multi-user hospital management system built with Streamlit. It provides role-based access for **Patients**, **Doctors**, and **Administrators** to manage hospital operations efficiently.

## ✨ Features

### 🔐 Role-Based Access
- **Admin**: Complete system control
- **Patient**: Personal healthcare management
- **Doctor**: Appointment & prescription management

### 🎯 Admin Capabilities
- ➕ Add new patients with auto-generated IDs
- 👥 View all registered patients
- 🔍 Search patients by ID
- 📊 Sort patients by age
- 🧾 Manage OPD queue
- 🛏️ Bed allocation and discharge
- 👨‍⚕️ Add new doctors
- 📈 View hospital statistics & analytics

### 👤 Patient Features
- 📋 View personal medical details
- 📅 Book appointments based on symptoms
- 💊 View prescriptions
- 📝 Update profile information
- 📚 Complete medical history
- 💰 Discharge and pay bills

### 👨‍⚕️ Doctor Features
- 📅 View scheduled appointments
- 💊 Add prescriptions to patient records
